## North Star: Epistemic Depth

This is not a homeschooling blog. This is an authority engine built at the intersection of three rare competencies:

| Competency | Contribution |
|---|---|
| **Molecular Biology** | Systems thinking - understanding knowledge as a network of interacting nodes |
| **Investigative Journalism** | Evidence synthesis - teaching learners to evaluate claims like a newsroom |
| **Instructional Design** | Learning architecture - Cognitive Load Theory, spiral curriculum, schema formation |

The site's differentiator is **Epistemic Depth**: the capacity to understand not just *what* to know, but *how* knowledge is constructed, justified, and revised.

## Core Methodologies

### Epistemic Depth

The 4-Layer architecture:

| Layer | Type | Project-as-Proof Modality |
|---|---|---|
| **I: Factual Foundation** | What | Complete a book |
| **II: Procedural Competence** | How | Craft work |
| **III: Conceptual Integration** | Why | Painting / visual art |
| **IV: Epistemic Agency** | How do we know | Cooking |

### Project-as-Proof Model

Epistemic depth was never assessed abstractly. Mastery was always demonstrated through a tangible project. The project is an **epistemic artifact** - proof that the learner has internalised knowledge deeply enough to transform it into something new in a different medium.

**The translation thesis:** Deep understanding enables medium-crossing. If you can only express knowledge in the same form you received it, you have memorised, not understood.

### STOBLS (Story-Based Learning Sequences)

A proprietary methodology developed over 15+ years of instructional design practice. Narrative as cognitive scaffold for complex concept acquisition.
